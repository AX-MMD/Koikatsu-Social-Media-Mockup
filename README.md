# Koikatsu Social Media Mockup
A tool to emulate social media posts

Access the tool online [here](https://ax-mmd.github.io/Koikatsu-Social-Media-Mockup/main.html) or download the [main.html file](https://raw.githubusercontent.com/AX-MMD/Koikatsu-Social-Media-Mockup/refs/heads/main/main.html) and open it in your browser. The work is entirely done localy and no content is sent online, the "uploaded" images are stored in your web browser.
